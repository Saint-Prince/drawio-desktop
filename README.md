# drawio-desktop

draw.io desktop app based on Electron

[![Build Status](https://travis-ci.org/jgraph/drawio-desktop.svg?branch=master)](https://travis-ci.org/jgraph/drawio-desktop)

To run this:
1. npm install
2. export NODE_ENV=development
3. npm start
